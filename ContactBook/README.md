# 📇 Contact Book - Task 5

A modern desktop Contact Book application built with Python and Tkinter. It allows users to efficiently store, manage, and search contacts through a clean and user-friendly interface.

## Features

- Add new contacts
- Edit existing contacts
- Delete contacts
- Search contacts by name, mobile number, or email
- View detailed contact information
- Duplicate mobile number validation
- Auto-generated contact avatars
- JSON-based data persistence

## Technologies Used

- Python
- Tkinter
- JSON

## How to Run

```bash
python ContactBook.py
```

## Project Goal

This project was developed to strengthen my understanding of GUI development, data validation, file handling, and application design using Python.
